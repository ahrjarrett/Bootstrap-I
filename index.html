Live • GitHub

Sun Ray Pools

Live • GitHub

Built website for a swimming pool client from scratch. Stack included Express.js for routing, Jade for the templating engine, Axios for making AJAX calls, Gulp for the dev workflow, and Bootstrap and Font Awesome for frontend work.

This project was fun, and a no-brainer excuse to work with the OpenWeatherMap API. Also, this project was my first real exposure to modularity and the MVC framework; you can explore the architecture in the app folder.
